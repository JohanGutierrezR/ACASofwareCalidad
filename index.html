ACA Calidad de Software

¿Qué es la Calidad de Software?

La calidad de software es el grado en que un producto satisface requisitos funcionales y no funcionales, y cumple expectativas de rendimiento, seguridad, mantenibilidad y experiencia de usuario. En entornos educativos la calidad asegura continuidad del aprendizaje, accesibilidad y confianza en la plataforma.

Medir para mejorar

Medir con métricas claras permite identificar debilidades, priorizar correcciones y demostrar mejoras. En educación, estas mediciones protegen la experiencia de docentes y estudiantes y reducen riesgos operacionales.

Criterios de Evaluación

A continuación, se describen los criterios clave utilizados para evaluar la calidad y el rendimiento de la aplicación educativa. Cada criterio le permite enfocar sus esfuerzos en diferentes aspectos de la calidad del software.

Guía rápida — Métodos de evaluación

INSTRUCCIÓN: Al seleccionar cada criterio se dirigirá a las herramientas y métodos de prueba asociados para la evaluación de la calidad de software.

Criterios y Herramientas de Prueba Rápida:

  • Funcionalidad: Postman (Pruebas de APIs y back-end).
  • Usabilidad: Lighthouse (a través de DevTools para análisis de Accesibilidad y UX).
  • Eficiencia: PageSpeed Insights (Medición de velocidad y Core Web Vitals).
  • Mantenibilidad: W3C Markup Validation Service (Validación de código HTML).
  • Portabilidad: Browserling (Pruebas Cross-Browser).
  • Seguridad: SSL Labs Server Test (Evaluación de certificado SSL/TLS y configuración del servidor).
  • Fiabilidad: SSL Labs Server Test (Estabilidad del certificado y conexión HTTPS).
  • Compatibilidad: Browserling (Simulación de la aplicación en diferentes entornos).

Metodología de Pruebas: Se sugiere una combinación de inspección funcional (checklists), pruebas con usuarios (usabilidad), pruebas de carga (eficiencia), revisión de seguridad y análisis estático del código (mantenibilidad).

Asignación de pesos: La plataforma permite editar el peso (porcentaje) de cada criterio para personalizar la puntuación final según las prioridades del proyecto.

Puntaje final = Σ (Valor del criterio × Peso asignado). La nota final va de 0 (mínimo) a 5 (máximo).
